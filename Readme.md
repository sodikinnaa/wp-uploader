# It Retceh API Post

Plugin WordPress untuk membuat post melalui **REST API** lengkap dengan:
- Validasi API Key
- Pemilihan penulis otomatis
- Upload featured image (dengan konversi ke WebP)
- Kategori dan tag otomatis
- Custom meta field
- Endpoint validasi plugin
