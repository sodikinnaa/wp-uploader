Plugin WordPress untuk membuat post melalui REST API lengkap dengan validasi API Key dan upload gambar WebP.

== Description ==
It Retceh API Post memungkinkan Anda membuat post di WordPress secara otomatis melalui endpoint REST API.
Plugin ini juga mendukung upload gambar format WebP, validasi API Key, serta kategori dan tag otomatis.

== Installation ==
1. Download file ZIP plugin ini.
2. Masuk ke WordPress Admin → Plugins → Add New → Upload Plugin.
3. Pilih file ZIP lalu klik **Install Now** → **Activate**.

== Frequently Asked Questions ==

= Bagaimana cara cek apakah plugin aktif? =
Gunakan endpoint berikut:
`/wp-json/buta-buku/v1/validasi-plugin`
